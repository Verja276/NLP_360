15.
Google Chrome (desktop)
- Click the three dots (⋮) → Settings → Privacy and security → Site Settings → Pop-ups and redirects.
- Toggle to “Allowed” or click Add and enter https://go.sfu.ca (or click the lock icon in the address bar → Site settings → Pop-ups and redirects → Allow).
- Refresh the page.

Firefox (desktop)
- Menu (☰) → Settings → Privacy & Security → Permissions → Block pop-up windows.
- Uncheck to disable blocking entirely or click Exceptions… and add https://go.sfu.ca.
- Refresh the page.

Microsoft Edge (desktop)
- Settings and more (…) → Settings → Cookies and site permissions → Pop-ups and redirects.
- Toggle to “Allowed” or click Add under Allow and enter https://go.sfu.ca.
- Refresh the page.

Safari (macOS)
- Safari → Settings/Preferences → Websites → Pop-up Windows.
- Find the current site and set to Allow, or set “When visiting other websites” to Allow to disable globally.
- Refresh the page.

Chrome on Android
- Chrome menu → Settings → Site settings → Pop-ups and redirects.
- Toggle to Allowed or add https://go.sfu.ca under Allowed.
- Refresh the page.

Safari on iPhone/iPad
- Settings app → Safari → toggle off Block Pop-ups.
- Return to Safari and refresh the page.

If enabling pop-ups doesn’t work, disable any ad‑blocker/extension temporarily and try again.
